# Whack-A-Note

Random-note fingerboard and rhythm trainer from Starlight Ukes.

## Timing pattern

Each session begins with one full count-in measure. In Root Mode, a note appears on beat 2; the player uses beats 3–4 to find it, then plays it on the following beat 1. The reference pitch sounds on that downbeat for self-checking.

In Root–Fifth Mode, the first root appears on beat 2 after the count-in. Play the root on the following beat 1 and the note a perfect fourth below on beat 2. The next root appears immediately on beat 3, giving you beats 3–4 to prepare for the next root–fifth pattern.

The metronome uses a synthesized wood-block sound, with a slightly lower and stronger sound on beat 1.

## Practice timer

The configurable practice timer defaults to 3 minutes. Enter a duration from 1–120 minutes, or enter 0 for an unlimited session. The timed portion begins after the count-in.
